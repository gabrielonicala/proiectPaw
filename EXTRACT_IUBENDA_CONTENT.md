# How to Extract iubenda Content

To use the static HTML approach:

1. Open the saved HTML file: `c:\Users\gabit\Desktop\Privacy Policy of 4319802 _ quillia.app.html`

2. Find the `<main>` tag (around line 88)

3. Copy everything from `<main>` to `</main>` (including those tags)

4. Save it as: `public/legal/privacy-policy-content.html`

5. The component will automatically:
   - Remove unwanted elements (headers, footers, buttons, etc.)
   - Apply your custom styling
   - Display it with your design

**Quick way:**
- Open the HTML file in a text editor
- Search for `<main>`
- Copy from `<main>` to `</main>` (inclusive)
- Paste into `public/legal/privacy-policy-content.html`

The styling will automatically match your existing privacy policy design!
